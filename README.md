# PRODIGY_GA_05

## Neural Style Transfer

### Objective

Apply the artistic style of one image to the content of another image using Neural Style Transfer.

### Tools Used

* Python
* Google Colab
* PyTorch
* Pillow
* Matplotlib

### Procedure

1. Upload a content image.
2. Upload a style image.
3. Apply style transfer.
4. Generate the stylized image.
5. Save and display the output.

### Input

* Content Image
* Style Image

### Output

A stylized image combining the content of one image and the artistic style of another.

### Result

Successfully applied neural style transfer and generated a stylized image.
